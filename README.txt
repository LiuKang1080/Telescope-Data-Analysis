1) Make sure that the telescope data file that needs to be analyzed (.dat file) is in the same directory as Telescope_Data_Analysis.py

2) Start Telescope_Data_Analysis.py by double clicking and enter the full data file    name (including extension) when prompted to ex. "frbs.dat" 

3) The program will do calculations for the highest elevation of an object and output a file called "FRBS_Finished_Calculation.dat". Within the file are the object's FRBName, Elevation and Azimuth.

4) After program is finished it will prompt you to press enter twice, This will close the program completely.
